# Keyboard Shortcuts

## Global Hotkey

Works from any app, even when Assistask is hidden.

| Shortcut | Action |
|---|---|
| `Ctrl + Shift + A` | Show / hide the Assistask window |

The hotkey has a 300ms debounce to prevent accidental double-triggers.

---

## In-App Shortcuts

These work when the task input field is focused.

| Shortcut | Action |
|---|---|
| `Enter` | Add task / run slash command |
| `↑` | Previous entry from input history |
| `↓` | Next entry from input history |
| `Escape` | Clear the input field |
| `Tab` | Autocomplete the highlighted slash command |

---

## Task Edit Mode

When editing an existing task (click the edit icon on a task):

| Shortcut | Action |
|---|---|
| `Enter` | Save changes |
| `Escape` | Cancel editing, discard changes |

---

## Tray

| Action | Result |
|---|---|
| Left-click tray icon | Show / hide window |
| Right-click tray icon | Open tray menu (Show/Hide, Quit) |
