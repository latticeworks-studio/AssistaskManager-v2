# Getting Started

## Installation

Download the latest `.msi` installer from the [Releases](https://github.com/latticeworks-studio/AssistaskManager-v2/releases) page and run it. Windows may show a SmartScreen prompt — click **More info → Run anyway**.

Assistask installs to your user directory and creates a tray icon on launch.

## Summon the Window

Assistask lives in your system tray. You can show or hide it three ways:

- **`Ctrl+Shift+A`** — global hotkey, works from any app
- **Left-click** the tray icon
- **Right-click** the tray icon → **Show / Hide**

The window is always-on-top by design — it floats over whatever you're working on so you can capture a task without losing focus.

Closing the window with the **×** button hides it to tray. To fully quit, right-click the tray icon → **Quit Assistask**.

## Adding Your First Task

Type naturally into the input bar at the bottom and press **Enter**.

```
review the proposal by end of week
```

Assistask reads what you typed, detects the due date and priority, and adds the task. A preview strip above the input shows what was detected before you commit.

No forms. No dropdowns. One step.

## The Five Views

Navigate between views with the **☰** menu button at the top left.

| View | What it's for |
|---|---|
| **Tasks** | Your main task list — add, complete, and manage tasks |
| **Morning Standup** | One-click summary of overdue, due today, and high-priority items — ready to paste into Slack or Teams |
| **Completed** | Browse finished tasks and permanently delete them |
| **Settings** | Timezone clocks, notifications, colors, data export |
| **About** | Version info |
